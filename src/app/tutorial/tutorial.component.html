Tutorial

  • Scrolling: Zoom in and out using your mouse-wheel
  • Dragging: Click and drag to move around
  • Alt + Scroll: Increases/Decreses Iterations
  • Shift + Scroll: Changes the current color mode
  • Ctrl + Scroll: Changes the current modifier mode
  • Right-click: Saves/Downloads image
  • Press j: Swaps between mandelbrot- and julia-view
  • Press m: Mutes the music
  • Press t: Opens/Closes the tutorial
  • Press Space: Shows/Hides the HUD
